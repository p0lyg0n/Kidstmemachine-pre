# Kids Timemachine

「一般社団法人スポーツタイムマシン」が進める新企画「Kids Timemachine」の企画検討用ビジュアライズツールです。3D空間上に3×5m床＋赤マット／テレビ列／センサーを設置して、構成の見た目・寸法・センサー配列を手軽に確認できます。

## 目的
- フロア・マット・センサーレイアウトの寸法感を把握しながら、TV配置やセンサーの視野を視覚的に確かめる
- センサー台数や表示の切り替え、寸法の読み取りが容易な簡易HUD付きでチーム内の共有資料として活用
- Webブラウザだけで動くため、プロトタイプ検証・企画説明資料の補足としても使いやすい

## 使い方
1. リポジトリをクローン・チェックアウトして、ルートに移動（例: `git clone ... && cd kidstimemachine`）。
2. `index.html` は単体で完結しているので、ローカルサーバーで配信するのが最も安定。例えば：
   ```bash
   python -m http.server 4173
   ```
   ブラウザで `http://localhost:4173/index.html` を開く。
3. 画面内のHUD（左上）でセンサー台数・可視化トグル・高さ/距離を調整。「操作」欄にある通り、ドラッグで回転、ホイールでズーム、右ドラッグで平行移動できます。
4. センサーは2台/4台に切り替え可能で、それぞれの視野とレイアウトを視覚的に確認できます。HUD下部の寸法テキストで数字を確認でき、パラメータも自動検証されています。

## 補足
- Three.js の CDN（esm.sh）を利用して依存を制御しているため、インターネット接続が必要です。
- センサー高さ・距離の調整は 50mm 刻みで、範囲や表示・非表示を HUD から切り替え可能です。
- テストはブラウザ起動時に自動実行され、失敗時はコンソール＋HUDのエラーメッセージで知らせます。
